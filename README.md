# BlackJack
Game of Blackjack, get drawn 2 cards by a dealer, and try not to bust out over 21!, try to get a higher value then the dealer without busting, and win big!

-Betting System (1.5 for winning) (2.0x for BlackJack)
-Hit Stand, or Split against the Dealer
-Winner Determination and Probability
-Colored Cards
-Infinite Play
